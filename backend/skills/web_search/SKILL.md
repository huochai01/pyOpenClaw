## 步骤
1. 当用户提出需要实时信息、最新数据或需要联网查询的问题时，使用此技能
2. 使用 `fetch_url` 工具访问 `https://www.baidu.com/s?wd={搜索关键词}`
3. 从返回的页面中提取搜索结果摘要
4. 用简洁自然的中文回复用户，并注明信息来源

## 注意事项
- 优先使用中文关键词进行搜索
- 对于复杂问题，可能需要提取多个搜索结果进行综合
- 搜索结果可能包含广告，需要识别并过滤
- 如果搜索结果不理想，可以尝试调整关键词重新搜索

## 示例
用户：今天有什么重大新闻？
助手：我来帮你搜索一下今天的重大新闻。
（调用 fetch_url 访问 https://www.baidu.com/s?wd=今天重大新闻）
